# My IT Support Homelab

A production-style Windows Server 2025 environment I built to develop hands-on
IT support and sysadmin skills. Currently job-hunting for entry-level helpdesk
roles in Sydney.

## What's in this repo

- 📘 [Runbooks](./runbooks) — Step-by-step guides for common support tasks
- 🎫 [Tickets](./tickets) — Simulated tickets with full diagnostic walkthroughs
- ⚙️ [Scripts](./scripts) — PowerShell automation for AD and diagnostics
- 📐 [Diagrams](./diagrams) — Network topology and AD structure
- 📅 [Weekly logs](./weekly-logs) — My learning journal, dated

## Lab environment

| Component | Details |
|---|---|
| Hypervisor host | HP EliteDesk 800 G6 SFF |
| CPU | Intel i7-10700  |
| RAM | 32GB DDR4 3200MHz |
| Storage | 1TB SATA SSD |
| OS (host) | Windows 11 Pro|
| Virtualisation | Hyper-V — VT-x enabled, UEFI, Secure Boot, TPM 2.0 |
| Domain Controller | Windows Server 2025 VM (DC01) — Hyper-V guest |
| Client VM | Windows 11 Pro VM — Hyper-V guest, domain-joined |
| Domain | nareshlab.local |
| Network | 192.168.x.x/24 (internal lab, Ethernet primary) |
